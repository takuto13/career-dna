## エスカレーション事項（たくとへ）

### 【推奨対応】type-strategy.html の301リダイレクト設定

type-strategy.html は旧バージョンのページで、現在はcanonical修正とrobots.txt Disallowで対応済み。
より確実にするため、Vercelのvercel.jsonにリダイレクト設定を追加することを推奨する。

以下の設定をvercel.jsonに追加するだけで対応できる（CTOが実装可能・たくとの承認を得てから実施する）：

```json
{
  "redirects": [
    {
      "source": "/type-strategy.html",
      "destination": "/type-strategist.html",
      "permanent": true
    }
  ]
}
```
