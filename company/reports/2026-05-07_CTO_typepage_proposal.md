## ページの目的

- SEOからの新規流入獲得（「戦略家タイプ 向いている仕事」「戦略家 転職 おすすめ」等の検索流入）
- 流入したユーザーをアフィリエイトリンクへ誘導してコンバージョン
- キャリアDNA診断への回遊促進

```

  h2: 戦略家タイプの特徴・強み

    （診断ページへのCTAセクション）
```

## レイアウト・デザイン提案

### 全体レイアウト

```
[ナビゲーション] ← index.html・result.html と共通
[ヒーローセクション]
  - 背景：薄いグレー or ブランドカラー（承認確認）
  - h1タイトル
  - タイプアイコン（🧭 または type-strategist.png）
  - リード文（100字程度）

[特徴・強みセクション]（白背景）
  - h2 見出し
  - チェックリスト形式の箇条書き

[向いている職種セクション]（薄いブルー背景 or 白）
  - 職種チップ（result.html の job-chips スタイル流用）

[向いていない職種・注意点セクション]（薄いアンバー背景）
  - caution-card スタイル流用

[アフィリエイトカードセクション]（白背景）
  - PRバッジ付き
  - jobs.html の ad-card スタイルをそのまま流用
  - 3枚カード縦並び（モバイル）

[診断CTAセクション]（ブランドカラー背景）
  - 「無料で診断スタート →」ボタン
  - 診断ページへの内部リンク

[フッター] ← 共通
```

### 既存スタイルの流用方針

| 要素 | 流用元 |
|---|---|
| ナビゲーション | result.html |
| ad-card | jobs.html |
| job-chips | result.html |
| caution-card | result.html |
| footer | 共通 |
| hero-cta ボタン | index.html |

→ 既存CSS（style.css）を最大限流用し、新規CSSは最小限に抑える方針。
