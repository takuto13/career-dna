# Issue #5 対応：CV・CTA強化施策提案

作成日：2026-06-28  
担当：CTO  
Issue：[#5 CV、CTAの強化施策を模索して](https://github.com/takuto13/career-dna/issues/5)

---

## 概要

現状の問題：FAQや情報量が多く、アフィリエイトリンク（転職エージェント広告）へのクリックが発生しにくい。  
目的：流入ユーザーを「診断 → エージェント登録 = 収益」の動線に乗せる。

**※ UIデザイン変更は流入データが取れてから実施（RULES.md方針）。本書は提案のみ。**

---

## 現状の収益動線（課題）

```
[診断ページ] → [タイプ結果ページ] → [jobs.html] → [エージェント登録]
                     ↑
                ここにFAQが大量にあり、ユーザーが
                読み込んで離脱してしまっている
```

---

## 施策提案

### 施策1：診断結果ページのCTA強化（最優先）

**現状：** タイプページの「おすすめ転職エージェント」セクションが下部にある  
**提案：** 診断結果を見た直後（Above the fold）にCTAカードを配置

```
【提案レイアウト案：タイプ結果ページ】
━━━━━━━━━━━━━━━━━
[タイプ名] ＜あなたは〇〇タイプ！＞
[強みの簡潔な説明 3行以内]
━━━━━━━━━━━━━━━━━
▼ あなたタイプにおすすめのエージェント
[エージェントカード1] [エージェントカード2] [エージェントカード3]
（ここに「無料登録する」ボタン）
━━━━━━━━━━━━━━━━━
[詳細説明・FAQ ↓ スクロールで見られる]
```

**期待効果：** エージェント登録CVRを現状比 2〜3倍に改善

---

### 施策2：FAQページのCTAインサート

**提案：** FAQ本文の中に定期的にCTAボックスを挿入

```
【FAQ 10問ごとに挿入するCTAボックス案】
─────────────────────
💡 あなたのキャリアタイプを無料診断中
「転職エージェントを選ぶ前に、まず自分のタイプを知ろう」
  [5分で無料診断する →] ← 診断ページへ
─────────────────────
```

**期待効果：** FAQページ滞在ユーザーの診断ページ回遊を促進

---

### 施策3：index.htmlのファーストビュー最適化

**現状：** トップページの診断ボタンが小さく、CVを促す文言が弱い  
**提案：**
- キャッチコピーを「転職・就活の答えは5分の診断にある」に強化
- 「無料で診断する」ボタンをファーストビューに大きく配置
- 「登録不要」「5分」「無料」の3要素を視覚的に強調
