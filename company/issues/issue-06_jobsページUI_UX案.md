# Issue #6 対応：求人ページ（jobs.html）UI/UX改善案

作成日：2026-06-28  
担当：CTO  
Issue：[#6 求人ページのCV向上のためのデザイン案思考](https://github.com/takuto13/career-dna/issues/6)

---

## 概要

jobs.htmlは広告（転職エージェントアフィリエイト）が下部に埋もれており、ユーザーが広告を見る前に離脱している。  
現在のブランディング・デザインを踏まえて、UI/UXを改善する提案を以下に示します。

**※ 実装はたくとの承認後。現在はデータ収集フェーズのため変更未実施（RULES.md方針）。**

---

## 現状の分析

### 問題点
1. **広告がページ下部に配置** → スクロールしないと見えない（多くのユーザーが離脱）
2. **広告とコンテンツの区別が不明** → どれが「クリックすべきもの」かわからない
3. **CTAボタンの視認性が低い** → 「登録する」ボタンが目立たない
4. **タイプ別フィルタがない** → 全エージェントが並んでいて選択しにくい

---

## 提案レイアウト案

### 案A：タイプ別フィルタ型（推奨）

```
[jobs.html 提案レイアウト]
━━━━━━━━━━━━━━━━━━━━━━━
【ヘッダー】あなたのタイプに合う転職エージェントを選ぼう
━━━━━━━━━━━━━━━━━━━━━━━
[タイプ選択タブ]
 戦略家 | 分析者 | 職人 | 調整役 | 挑戦者 | 表現者 | 守護者 | 育成者
━━━━━━━━━━━━━━━━━━━━━━━
[選択タイプ別 おすすめエージェントカード × 3枚]

┌─────────────────────┐
│ ★★★ リクルートエージェント   │
│ 求人数No.1 ・ 20万件以上     │
│ [〇〇タイプにおすすめの理由]   │
│       [無料で登録する ▶]     │
└─────────────────────┘

┌─────────────────────┐
│ ★★☆ doda                  │
│ エージェント + 求人サイト両方対応 │
│ [〇〇タイプにおすすめの理由]   │
│       [無料で登録する ▶]     │
└─────────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━
【その下に】全エージェント比較表（スクロールで閲覧可）
━━━━━━━━━━━━━━━━━━━━━━━
```

**メリット：**
- 診断済みユーザーが「自分のタイプ」を選ぶだけで最適解が見つかる
- ファーストビューにエージェントカードが来るのでクリック率UP
- タイプ選択が「自己確認」になり滞在時間・エンゲージメントUP

## ブランディング・デザイン方針

### 現在のブランドカラー：
- プライマリ：紺系（#1e3a5f 相当）
- アクセント：各タイプカラー（紫・青・緑系）

### UI/UX改善の方向性：
1. **清潔感・信頼感** → 白背景ベース、情報を絞る
2. **エージェントカードのCTAボタン** → タイプカラーに合わせた色にして視認性UP
3. **「無料登録」の文言** → 「無料・1分で登録」「5分の登録で転職成功率が上がる」等に変更
4. **ソーシャルプルーフ** → 「転職者の〇〇%が複数エージェントを利用」等のコピーを追加
