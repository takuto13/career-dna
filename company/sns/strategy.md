## 1. ターゲット分析

### メインターゲット
**22〜28歳の就活生・第二新卒・転職検討中の会社員**

| 属性 | 詳細 |
|---|---|
| 年齢 | 22〜28歳（コア）、29〜35歳（サブ） |
| 状況 | 就活中 / 入社1〜3年目で転職検討 / 現職に違和感を持ち始めた会社員 |
| 心理 | 「自分に向いてる仕事が本当にわからない」「このままでいいのか不安」 |
| X上の行動 | 企業情報収集（62%）/ 就活生同士の情報交換 / #就活生と繋がりたい |
| 反応するコンテンツ | 共感系・有益情報・自己分析系・診断系 |

### ターゲットが抱える3つの不安
1. **自分に向いてる仕事がわからない** → キャリアDNAで解決できる
2. **周りと比べて自分だけ遅れている気がする** → 共感で刺さる
3. **転職・就活で何から始めればいいかわからない** → 有益情報で信頼を得る

## 2. アカウントコンセプト

**「自分らしい仕事の見つけ方を、一緒に考えるアカウント」**

- トーン：親しみやすいが、ちゃんと役に立つ
- ポジション：診断ツールの発信元 ではなく「キャリアに悩む人の相棒」
- 差別化：MBTI・16Personalities は性格分析。キャリアDNAは**仕事への活かし方**に特化

## 3. バズ設計（診断コンテンツの拡散構造）

診断コンテンツがシェアされる理由は「自己表現欲求」。
「自分のタイプを知ってほしい」という心理を最大限に活用する。

### シェアを誘発する3要素
| 要素 | キャリアDNAでの実装 |
|---|---|
| 自分だけの結果 | 8タイプのパーソナライズ結果 |
| 意外性 | 「え、こんな弱点あるの？」と思わせる率直な分析 |
| シェアしたくなる | 「あなたは何タイプ？」導線 + #キャリアDNA診断 |

### シェア文言の方向性（タイプ別で自動生成）
```
「キャリアDNAで診断したら「戦略家タイプ」だった。
確かに一人で黙々と考える系の仕事が好き。
あなたは何タイプか試してみて↓
https://career-dna.jp #キャリアDNA診断」
```
→ 「あなたは何タイプ？」の一言が連鎖的な診断を生む

## 4. 投稿戦略

### 週間カレンダー

| 曜日 | 型 | 具体例 |
|---|---|---|
| 月 | **共感・あるある** | 「分析タイプの人あるある5選」 |
| 火 | **有益情報・数字** | 「戦略家タイプが向いてる職種TOP5、調べてみた」 |
| 水 | **問いかけ・参加型** | 「仕事でいちばんストレスになること、どれ？」（投票機能活用） |
| 木 | **深掘り・教育** | 「自己分析で失敗する人の共通点3つ」 |
| 金 | **診断誘導CTA** | 「週末2分で仕事タイプわかります。25問だけ」 |
| 土 | **比較・話題便乗** | 「MBTIとキャリアDNAの違いをまとめてみた」 |
| 日 | **まとめ・振り返り** | 「今週のコメントで多かった悩みに答えます」 |

### バズ狙い投稿（月2〜3本）
- 「このタイプの人だけわかる」シリーズ
- 「診断したら意外なタイプだった人いる？」
- 「就活生100人に聞いた、本当に向いてる仕事の見つけ方」
- 「◯◯な仕事が向いてる人の特徴、全部当たってた」

## 5. アルゴリズム攻略（2026年版）

| ルール | 実装 |
|---|---|
| 投稿後30分以内に自己リプライ | 補足情報 or 「あなたは何タイプ？」を追記 |
| リプライへの返信を必ず行う | 24時間以内に返信。会話を続ける |
| 外部リンクは本文に入れない | リンクはリプライに。本文はテキストのみ |
| 投稿時間 | 平日7〜9時 / 20〜22時、休日14〜17時 |
| 毎日最低1投稿 | 橘結衣が週7本を事前に用意 |
| 投票機能を週1で使う | エンゲージメント稼ぎに最も効率的 |
