**Q290：転職で空白期間・ブランクがある場合のタイプ別面接説明術と採用担当者への伝え方**
- 狙いKW：「転職 空白期間 ブランク 面接 説明 タイプ別」「転職 ブランク 何ヶ月 不利 対策」「転職 空白期間 理由 言い方 採用担当 好印象」
- 収益直結：空白期間が不安な転職者はエージェント活用率が高い → ビズリーチ・JAC誘導必須。jobs.html?type=XXXリンク必須
- 内容ポイント：
  - 空白期間の評価基準（3ヶ月以内は不問・4〜6ヶ月は要説明・6ヶ月以上は計画性の説明必須）
  - タイプ別の伝え方：戦略家/分析者→スキルアップ・自己投資訴求型（「この期間で○○を身につけた」） / 職人→専門スキル磨き・資格取得強調型 / 調整役/育成者→家族ケア・誠実スタンス型（「家族のために選んだ決断」） / 挑戦者/表現者→新挑戦準備・自己発見型 / 守護者→慎重な意思決定・次職場への熱意型
  - 「空白期間を強みに変える」フレームで締める → jobs.html誘導

**Q291：転職の適性検査（SPI・CAB/GAB）対策とタイプ別の得意・不得意・スコアアップ法**
- 狙いKW：「転職 SPI 対策 タイプ別 得意 不得意」「転職 適性検査 CAB GAB 苦手 タイプ診断」「SPI 言語 非言語 タイプ別 強み 弱み 克服法」
- 収益直結：適性検査で困っている転職者はエージェント相談ニーズが高い → jobs.html誘導
- 内容ポイント：
  - タイプ別SPI強み：戦略家/分析者→非言語推論・論理問題が得意 / 職人→処理速度・正確性が強み / 表現者/挑戦者→言語問題・読解が得意 / 守護者/調整役→コツコツ反復練習で伸びやすい
  - CAB/GAB対策はエンジニア系（分析者・職人タイプ）に重要
  - 適性検査前にエージェント登録でアドバイスを受け取ることが効果的 → jobs.html誘導

**Q292：転職活動にかかる費用・期間コストのタイプ別管理法と無駄ゼロの転職活動術**
- 狙いKW：「転職 費用 いくら かかる タイプ別 管理」「転職活動 期間 何ヶ月 コスト タイプ別 最短」「転職 費用 スーツ 写真 交通費 節約」
- 収益直結：「費用が心配」層をエージェント無料サービスへ誘導 → jobs.html CTA
- 内容ポイント：
  - 転職活動の平均費用（スーツ・写真・交通費・書類費）：合計3〜10万円が相場
  - タイプ別費用管理：戦略家/分析者→ROI重視で費用対効果を計算して優先投資 / 職人→最小コストで最大効果（オンライン面接活用） / 挑戦者→費用惜しまず応募数勝負 / 守護者/調整役→慎重に費用計画を立てて進める
  - エージェント利用は完全無料（企業負担）→ jobs.html誘導で「費用ゼロ」の安心感を提供

## 背景・理由

- **Q290（空白期間・ブランク）**：JAC・マイナビ・リクルートエージェントなど主要競合が空白期間コンテンツを掲載するが、「タイプ別の説明の仕方」は完全競合ゼロ（2026-07-01競合調査で確認）。空白期間を抱える転職者のエージェント依存度が高く、ビズリーチ・JACへのCVR最大化コンテンツ
