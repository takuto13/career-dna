## タスク1：faq.html 検索機能実装 + カテゴリ別5問+CTA構成（Issue #9 承認済み）

### 背景・理由

2026-06-30 たくとコメント（Issue #9）にて以下が明示承認された：

> 「FAQページには検索機能を実装して項目が多くても知りたいことがすぐ見つかるようにして。カテゴリ名→FAQピックアップ5問→もっと見る→CTAを項目分」

現状（319問が単純一覧）では：
- ユーザーが目的のFAQを見つける前に離脱
- CTAが下部にしかなく収益動線が弱い
- 離脱率高→アフィリエイトCTRゼロに近い

改善後：
- 検索で即座に目的FAQへアクセス
- 各カテゴリ末尾にCTAを配置→収益動線を7箇所確保
- ユーザー滞在時間増加→SEOシグナル改善

- 現在のFAQリストから選択（新規追加ではなく既存問の選抜）

### 実装内容C：JavaScript「もっと見る」toggle

```javascript
function toggleMore(id) {
  const el = document.getElementById(id);
  const btn = el.previousElementSibling;
  if (el.style.display === 'none') {
    el.style.display = 'block';
    btn.textContent = btn.textContent.replace('▼', '▲').replace('もっと見る', '閉じる');
  } else {
    el.style.display = 'none';
    btn.textContent = btn.textContent.replace('▲', '▼').replace('閉じる', 'もっと見る');
  }
}
```
