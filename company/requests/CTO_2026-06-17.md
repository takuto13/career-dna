**Q191：「転職活動 うまくいかない 原因 改善 対処法 モチベーション タイプ別 立て直し方」**

背景：転職活動を始めて1〜2ヶ月経った人が「なぜ書類選考が通らないのか」「面接で落ち続ける理由は何か」を検索する高CVRフェーズ。7月スパイク期は転職活動スランプに陥る人が急増するタイミング。キャリアDNA独自のタイプ別「うまくいかない原因と立て直し法」は競合がほぼ対応していないブルーオーシャンKW。

回答骨子：
- 転職活動がうまくいかない主な原因4つ：①自己分析が浅く強みを言語化できていない②応募先の業界・職種がタイプと合っていない③職務経歴書の訴求ポイントがズレている④タイプに合わないエージェントを選んでいる
- タイプ別のよくある「うまくいかない理由と立て直し法」（戦略家：論理的すぎて感情的な面接官と噛み合わない→エピソードに数字と感情を両方入れる / 職人：専門性を前面に出しすぎて汎用性がないと判断される→幅広い応用可能性をアピール / 調整役：強みが薄く見られがち→「チームを変えた具体エピソード」で差別化 / 挑戦者：在籍期間の短さを問われる→「挑戦の結果」を数値で示す / 表現者：ポートフォリオが弱い→作品数より質と背景ストーリーを充実させる / 守護者：安定志向が保守的に映る→「守ることで組織を成長させた実績」をアピール / 育成者：育成成果が数字化されていない→「育成した人数・成長率」を定量化する / 分析者：データ分析の成果を伝えられていない→「分析が業績に貢献した実績」を数値化する）
- 「まずはタイプを確認してから、タイプに合ったアピール法を選ぶことが最短の立て直し策」→診断ページ（career-dna.jp）誘導
- CTA：jobs.html（タイプパラメータ付き）への内部リンク（タイプ別エージェント再選択）

**Q192：「転職活動 書類選考 通過率 上げる コツ 職務経歴書 通らない 原因 改善策 2026」**

背景：「書類選考が全然通らない」は転職活動中の最大の悩みの1つ。検索ボリュームが高く、エージェントに職務経歴書を見てもらうことへの誘導（jobs.html）に直結する最高収益KW。競合は一般的なTipsを並べるだけだが、キャリアDNAはタイプ別の書類作成法でFAQ深度の差別化が可能。

回答骨子：
- 書類選考通過率を上げる3ステップ：①タイプに合った「強みの切り口」を見つける②実績を数字で表現する（「〇〇を改善した」→「〇〇を△%改善し、売上×円増加に貢献した」）③応募先企業の求めるスキルとタイプの強みを接続する
- タイプ別の書類で効く「強みの切り口」（戦略家：論理的課題解決の実績・数字の改善率 / 分析者：データ分析が業績に繋がった具体事例 / 職人：専門スキルの深さと再現性 / 調整役：チームを動かした具体的エピソード / 挑戦者：困難な状況を打開した行動力 / 表現者：ポートフォリオとクリエイティブの成果 / 守護者：正確性・信頼性が組織に貢献した実績 / 育成者：部下・後輩を育てた成果の数値化）
- 「転職エージェントに職務経歴書を見てもらうと、担当者が書類をタイプの強みに合わせてブラッシュアップしてくれる。無料なので活用しない手はない」
- CTA：jobs.html（タイプパラメータ付き）への内部リンク（アフィリエイト収益直結）

**Q193：「転職 最終面接 通過率 合格 コツ 最終面接 落ちる 原因 対策 タイプ別」**

背景：最終面接まで進んだ人が「最終面接で落ちる理由と対策」を検索する高CVR層。7月スパイク期は書類→面接→最終面接のフローが加速するタイミング。「なぜ御社か」「5年後のビジョン」の語り方がタイプ別に異なる切り口は競合未対応。

回答骨子：
- 最終面接で落ちる主な理由3つ：①「なぜうちの会社か」が弱い②長期キャリアビジョンが不明確③現場と経営層（最終面接官）へのアピールポイントがズレている
- タイプ別の最終面接で刺さるアピール法（戦略家：5年後のキャリアビジョンを「経営視点」で語る / 分析者：データと論理で「この会社でできること」を具体化 / 職人：「この会社でしか磨けない専門性」を強調 / 調整役：「このチームと一緒にやりたい理由」を人間関係視点で伝える / 挑戦者：「この会社で達成したい挑戦」を具体的な数字目標で語る / 表現者：「この会社で作り出したいもの・世界観」をビジョンで語る / 守護者：「この会社の信頼性・安定性を守りながら成長に貢献する」を伝える / 育成者：「このチームを育て組織を強くする」という貢献像を具体化）
- 「最終面接前に自分のタイプを再確認すると、面接官への刺さるポイントが明確になる」
- CTA：career-dna.jp + jobs.html（面接コーチング可能なエージェント誘導）

## タスク2：全8タイプページ FAQ 57問目追加（書類選考通過率・タイプ別職務経歴書の書き方系KW）

### タイプ別回答骨子（全8タイプ）

- **戦略家**：職務経歴書で最も効果的な強みの切り口は「戦略立案→数値結果」の一本線。①課題設定（市場・組織・業務の問題を特定した）②戦略構築（解決策を設計した）③実行と結果（〇%改善・△円創出）の3段構成が最高評価。書き方のコツ：「論理的思考プロセスが見える実績記述」を意識する。書類選考を突破したあとはjobs.html（?type=strategist）でハイクラス転職エージェントと職務経歴書をさらに磨く。

- **分析者**：職務経歴書の最強フォーマットは「データ×業績貢献の数式化」。①分析したデータの規模・種類②分析から導き出したインサイト③そのインサイトが組織の意思決定・業績にどう繋がったかを数値で示す。「数字を扱った実績」が1つあるだけで書類通過率が大幅に変わる。jobs.html（?type=analyst）でデータ職種特化エージェントと連携するとさらに強化できる。

- **職人**：職務経歴書で職人タイプが最も評価される切り口は「専門性の深さと再現性」。①習得した専門スキルの具体的な技術・ツール②そのスキルを使って解決した問題と結果③同じ専門性を新しい会社で再現できることを示す。応用可能性も併せて示すことで「汎用性がない」という懸念を払拭できる。jobs.html（?type=craftsman）で専門職特化エージェントに書類を見てもらうとミスマッチ採用が防げる。

- **調整役**：職務経歴書で調整役タイプが差別化できる切り口は「チームを動かした具体エピソード」。①誰に対して、何の目標に向けて②どのように調整・ファシリテーションしたか③チームとしての成果を数値で示す。「私が動かした」ではなく「チームが動けるように私が設計した」という語り方が最高評価を生む。jobs.html（?type=mediator）で人材系・HR系エージェントに書類を磨いてもらうと効果的。

- **挑戦者**：職務経歴書で挑戦者タイプの評価を最大化するのは「困難→行動→突破口」のエピソード型記述。①誰もやらなかった・難しいと言われた課題②自分が主体的に動いたアクション③結果として出た成果（数字）。転職回数が多い場合は「各社での挑戦の結果」を並べることで「挑戦者としての一貫性」に変換できる。jobs.html（?type=challenger）でベンチャー・成長企業特化エージェントに書類を見てもらう。

- **表現者**：職務経歴書よりもポートフォリオが評価の中心になるタイプ。書類ではポートフォリオの「誰のために・何のために・どんなコンセプトで作ったか」のストーリーを短く語ることが最重要。実績は「再生数」「受賞歴」「クライアントの反応」で数値化する。書類選考通過のカギは「ポートフォリオへの誘導リンクを職務経歴書に明記すること」。jobs.html（?type=creator）でクリエイティブ特化エージェントと相談するのが最短ルート。

- **守護者**：職務経歴書で守護者タイプが最も評価される切り口は「正確性・信頼性が組織に貢献した実績」。①担当した業務範囲の精度・品質②ミスゼロ・コンプライアンス遵守・リスク管理の具体的な取り組み③それが組織の信頼性向上・損失防止にどう貢献したか。「几帳面すぎる」という見方を「組織の安定に不可欠な人材」に転換する記述が重要。jobs.html（?type=guardian）で公務員・医療・福祉系特化エージェントに書類を確認してもらう。

- **育成者**：職務経歴書で育成者タイプが評価される最強の切り口は「育てた人数・育成した結果の数値化」。①指導・育成した人数・期間②育成方法の具体的なアプローチ③育成された人の成長結果（昇格数・スキル向上・業績改善）。「私が育てた後輩が△人昇格した」「チームの残業時間を○時間削減した（育成で業務効率UP）」などの数値が書類選考で強く機能する。jobs.html（?type=nurturer）で人材育成・研修・教育系エージェントに書類を磨いてもらう。
