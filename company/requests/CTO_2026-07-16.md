## タスク1：index.html 構造化データ強化（Organization + WebApplication JSON-LD）

**背景：** Google検索でのブランド認知・サイテーション獲得のためには Organization マークアップが有効。AI検索（Perplexity・ChatGPT）からの参照獲得にも効果的。

**実装内容：**

index.htmlの既存JSON-LDに以下を追加（既存の `<script type="application/ld+json">` タグの後に新しいタグとして追加）：

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "@id": "https://career-dna.jp/#organization",
  "name": "キャリアDNA",
  "url": "https://career-dna.jp",
  "description": "AI搭載の無料キャリア診断サービス。25問・5分・登録不要で8タイプのキャリア診断を提供。転職・就活のための自己分析を即座に実施できる。",
  "sameAs": [],
  "logo": {
    "@type": "ImageObject",
    "url": "https://career-dna.jp/favicon.ico"
  }
}
```

**確認ポイント：** 既存のWebApplication JSON-LDと重複しないように

## タスク2：全8タイプページ → jobs.html 内部リンクのアンカーテキスト最適化

**背景：** 各タイプページからjobs.htmlへのリンクアンカーテキストが「転職求人を探す」など汎用的な表現になっている場合、SEO的なキーワード強化の機会あり。タイプ固有のキーワードをアンカーに入れることで内部リンクのSEO価値が向上する。

**実装内容：**

各タイプページの jobs.html へのリンクアンカーテキストを以下のように更新：

| タイプページ | 現在のアンカー（想定） | 推奨アンカー |
|------------|---------------------|------------|
| type-strategist.html | 転職求人を探す | 戦略家タイプにおすすめの転職先を見る |
| type-analyst.html | 転職求人を探す | 分析者タイプにおすすめの転職エージェント |
| type-craftsman.html | 転職求人を探す | 職人タイプに向いている仕事・転職先 |
| type-mediator.html | 転職求人を探す | 調整役タイプにおすすめの転職エージェント |
| type-challenger.html | 転職求人を探す | 挑戦者タイプに向いている転職先を見る |
| type-creator.html | 転職求人を探す | 表現者タイプのおすすめ求人・転職先 |
| type-guardian.html | 転職求人を探す | 守護者タイプにおすすめの転職エージェント |
| type-nurturer.html | 転職求人を探す | 育成者タイプに向いている転職・職種 |

**注意：** デザイン変更は禁止。テキスト変更のみ。既存リンクのhref先（jobs.html?type=XXX）は変更しない。
