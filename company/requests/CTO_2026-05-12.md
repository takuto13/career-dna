## タスク1：全8タイプページ 内部クロスリンク追加

### 背景・理由

現在、各タイプ詳細ページ（type-craftsman.html 等）は互いにリンクしていない。
Googleはサイト内の内部リンク構造でページの重要度を判断する。現状ではタイプページ間のリンクジュースが分断されており、SEO的に損をしている。

RULES.md では「内部リンクの追加・改善」は自律実行OK。**UI・デザイン変更ではなく、SEOテキスト改善として実施すること。**

### 期待する成果物

各タイプ詳細ページ（8ページ）の末尾（アフィリエイトカードセクションの後）に、他タイプへのリンクセクションを追加する。

**実装仕様：**
- セクション見出し：`<h2>他のキャリアタイプも見てみる</h2>`（または同等の見出し）
- 各タイプへのテキストリンクを8タイプ分並べる（自分のタイプを除いた7本 + 自分のタイプはリンクなし or 先頭に自タイプ明記）
- 既存ページのCSSクラス・スタイルを流用（新規スタイル追加は最小限）
- リンクアンカーテキスト例：`戦略家タイプを見る` / `職人タイプを見る` 等

**対象ファイル（8ページ）：**
```
public/type-strategist.html  → 戦略家タイプ
public/type-analyst.html     → 分析者タイプ
public/type-craftsman.html   → 職人タイプ
public/type-mediator.html    → 調整役タイプ
public/type-challenger.html  → 挑戦者タイプ
public/type-creator.html     → 表現者タイプ
public/type-guardian.html    → 守護者タイプ
public/type-nurturer.html    → 育成者タイプ
```

### 注意事項

- デザインに大きな変更が必要な場合は実装せずに提案書を作成してたくとへ確認を求めること
- 既存のアフィリエイトカード・FAQセクションは変更しない
- sitemap.xml の各タイプページの lastmod を本日日付（2026-05-12）に更新すること

## タスク2：jobs.html コンテンツSEO強化

### 背景・理由

jobs.html はアフィリエイト収益に直結するページ。競合調査では「タイプ別おすすめエージェント」CTAが最も収益直結する導線と判明している。しかし現状のjobs.htmlはSEOコンテンツが薄い可能性がある。

「転職エージェントおすすめ」「キャリア診断 転職」等のキーワードで流入できれば、アフィリエイトクリックに直結する。

RULES.md では「流入に関わるテキスト修正（SEOキーワード追加・ページタイトル・meta・見出し）」は自律実行OK。

### 期待する成果物

#### title タグの最適化

```html
<title>診断結果別おすすめ転職・就活サービス一覧 | キャリアDNA</title>
```

#### meta description の最適化

```html
<meta name="description" content="キャリアDNA診断結果に基づく、おすすめ転職エージェント・就活サービス一覧。あなたのキャリアタイプに合ったサービスで理想の転職・就活を実現しよう。無料・登録不要で診断可能。">
```

#### H1/H2 見出しのキーワード最適化

- H1：「あなたのキャリアタイプに合った転職・就活サービス」等、タイプ診断×転職サービスの連動を訴求
- H2：サービスカテゴリ別に「転職エージェント（20代・第二新卒向け）」「IT・エンジニア転職」「就活支援サービス」等の見出しを設ける

#### 各サービス説明文へのキーワード追加

各アフィリエイトサービスの説明文に「20代転職」「未経験転職」「年収アップ」「ITエンジニア転職」等の具体的キーワードを自然挿入する。

#### sitemap.xml 更新

- jobs.html の lastmod を `2026-05-12` に更新

### 注意事項

- UIデザイン・レイアウト変更は行わない（テキスト・head内のみの改善）
- 既存のアフィリエイトリンクのURLは変更しないこと（トラッキングURLを壊さない）
- デザイン変更が必要な場合は提案書を作成してたくとへ確認を求めること
