### タスク1：WebSite JSON-LD + SearchAction を index.html に追加【最優先】

**背景・理由**
現在 index.html には FAQPage・Speakable JSON-LD は実装済みだが、サイト全体を表す `WebSite` スキーマが未実装。`SearchAction` を付与することで Google のサイトリンク検索ボックス取得・ナレッジパネルのエンティティ認識強化・LLMO対策になる。ChatGPT / Perplexity 等のAI検索エンジンもサイト全体の概要を `WebSite` スキーマで認識することが多い。

**実装内容**

index.html の `<head>` 内（既存のJSON-LD `<script>` タグの直後）に以下を追加：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "@id": "https://career-dna.jp/#website",
  "name": "キャリアDNA",
  "alternateName": ["Career DNA", "AIキャリア診断 キャリアDNA"],
  "description": "AIが25問で強み・価値観・向いている仕事タイプを診断する無料AIキャリア診断。登録不要・5分で完了。",
  "url": "https://career-dna.jp/",
  "inLanguage": "ja",
  "publisher": {
    "@type": "Organization",
    "@id": "https://career-dna.jp/#organization",
    "name": "キャリアDNA",
    "url": "https://career-dna.jp/"
  },
  "potentialAction": {
    "@type": "SearchAction",
    "target": {
      "@type": "EntryPoint",
      "urlTemplate": "https://career-dna.jp/index.html"
    },
    "query-input": "required name=search_term_string"
  }
}
</script>
```

また、既存の Organization JSON-LD（about.html に実装済み）を index.html にも追加する：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "@id": "https://career-dna.jp/#organization",
  "name": "キャリアDNA",
  "url": "https://career-dna.jp/",
  "logo": "https://career-dna.jp/ogp.png",
  "sameAs": [
    "https://twitter.com/career_dna_jp"
  ],
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "customer support",
    "url": "https://career-dna.jp/contact.html"
  }
}
</script>
```

**期待成果物**
- index.html に WebSite + Organization JSON-LD 追加済み
- sitemap.xml の index.html lastmod を 2026-05-16 に更新
- コミット・プッシュ完了

**注意事項**
- 既存の FAQPage・Speakable JSON-LD は削除・変更しないこと
- デザイン・見た目への変更は不要（head内のみの変更）

### タスク2：全タイプページ → faq.html・about.html への内部リンク追加

**実装内容**

全8タイプページ（type-craftsman.html / type-strategist.html / type-mediator.html / type-challenger.html / type-analyst.html / type-creator.html / type-guardian.html / type-nurturer.html）の **アフィリエイトカードセクションの直後（または既存フッター直前）** に以下のナビリンクブロックを追加：

```html

<div style="text-align:center; margin: 2rem 0; padding: 1rem; border-top: 1px solid #e5e7eb;">
  <p style="font-size:0.85rem; color:#6b7280; margin-bottom:0.75rem;">キャリアDNAについて詳しく</p>
  <nav style="display:flex; justify-content:center; gap:1.5rem; flex-wrap:wrap;">

  </nav>
</div>
```

**期待成果物**
- 全8タイプページに上記ナビリンクブロック追加済み
- sitemap.xml の各タイプページ lastmod を 2026-05-16 に更新
- コミット・プッシュ完了

**注意事項**
- スタイルはインラインで最小限。デザイン変更にならないよう、既存のデザイントーン（グレー・インディゴ系）に合わせること
- 既存の「他のタイプを見る」グリッドは削除・変更しないこと

### タスク3：jobs.html に ItemList JSON-LD 追加

**背景・理由**
jobs.html（おすすめ転職・就活サービス一覧ページ）は現在 WebPage JSON-LD のみ。掲載しているサービス一覧を `ItemList` スキーマで構造化することで、Google の検索結果でリッチスニペット（リスト表示）が取得できる可能性がある。アフィリエイト流入の増加に直結。

**実装内容**

jobs.html の `<head>` 内に以下を追加（掲載サービス名は現在 jobs.html に掲載中のものを確認して実際の名称・URLに合わせること）：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "ItemList",
  "name": "キャリアタイプ別おすすめ転職・就活サービス一覧",
  "description": "診断で分かったキャリアタイプに合った転職エージェント・就活サービス",
  "url": "https://career-dna.jp/jobs.html",
  "numberOfItems": 8,
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "doda",
      "url": "https://career-dna.jp/jobs.html#doda",
      "description": "転職者向け・求人数No.1クラスの総合転職エージェント"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "マイナビ転職",
      "url": "https://career-dna.jp/jobs.html#mynavi",
      "description": "20代・第二新卒の転職に強い転職サービス"
    }
  ]
}
</script>
```

※ 掲載サービス数・名称・説明は現在の jobs.html の内容に正確に合わせること。上記は例示のみ。

**期待成果物**
- jobs.html に ItemList JSON-LD 追加済み
- sitemap.xml の jobs.html lastmod を 2026-05-16 に更新
- コミット・プッシュ完了

**注意事項**
- jobs.html を開いて掲載中のサービス一覧を確認してから ItemList を作成すること（サービス数・名称を正確に反映）
- デザイン変更不要（head内のみの変更）
