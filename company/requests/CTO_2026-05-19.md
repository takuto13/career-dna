### タスク3：about.html に Person JSON-LD 追加（E-E-A-T強化）

**背景・理由**
Googleの「E-E-A-T（経験・専門性・権威性・信頼性）」評価において、サイト運営者の情報が構造化データとして明示されていると有利。about.htmlには運営者情報（たくと）が記載されているが、Person JSON-LDがない。これを追加することで：
1. Googleが「このサイトの運営者は誰か」を機械的に理解できる
2. 著者としての権威性強化・LLMO（AI検索参照）での引用確率向上
3. 費用ゼロ・実装10分・効果：中

**対象ファイル：** `public/about.html`

**実装内容**
about.htmlの `<script type="application/ld+json">` ブロックに Person スキーマを追加（既存JSON-LDの後に追加）：

```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "たくと",
  "url": "https://career-dna.jp/about.html",
  "jobTitle": "キャリアDNA 運営者",
  "worksFor": {
    "@type": "Organization",
    "name": "キャリアDNA",
    "url": "https://career-dna.jp"
  },
  "sameAs": [
    "https://twitter.com/career_dna_jp"
  ]
}
```

※ Xアカウント（`sameAs`）のURLは正しいものを確認してから記載すること。不明な場合は `sameAs` フィールドを省略してよい。

**期待成果物**
- about.htmlにPerson JSON-LD追加済み
- sitemap.xml の about.html lastmod を 2026-05-19 に更新（変更した場合）
- コミット・プッシュ完了

**注意事項**
- 既存のOrganization JSON-LDや他のJSON-LDと重複しないよう確認すること
- HTML構造は変更しないこと（JSON-LD scriptタグの追加のみ）
