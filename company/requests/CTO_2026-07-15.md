## タスク2：SearchAction JSON-LD の追加（index.html）

**目的：** Googleのリッチリザルト（サイト検索ボックス）への対応・LLMO強化

**実装内容：**
`index.html` の `<head>` 内にある既存 `<script type="application/ld+json">` に続けて以下を追加：

```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "url": "https://career-dna.jp/",
  "name": "キャリアDNA",
  "description": "AI診断で向いてる仕事・キャリアタイプを5分で判明。転職・就活に特化した無料診断。",
  "potentialAction": {
    "@type": "SearchAction",
    "target": {
      "@type": "EntryPoint",
      "urlTemplate": "https://career-dna.jp/faq.html?q={search_term_string}"
    },
    "query-input": "required name=search_term_string"
  }
}
```
