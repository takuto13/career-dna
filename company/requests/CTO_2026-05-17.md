### タスク2：jobs.html に ItemList JSON-LD 追加【繰り越し】

**背景・理由**
jobs.html（おすすめ転職・就活サービス一覧ページ）は現在 WebPage・FAQPage JSON-LD のみ実装済み。掲載サービスを `ItemList` スキーマで構造化することでGoogleのリッチスニペット（リスト表示）取得の可能性が高まる。アフィリエイトクリック流入の増加に直結する重要施策。

**実装内容**

jobs.html を開いて掲載中サービスの名称・説明を確認し、正確な `ItemList` JSON-LDを作成して `<head>` 内に追加する（既存JSON-LDの後ろ）。

以下はフォーマット例（サービス数・名称・説明は必ず実際のjobs.html内容に合わせること）：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "ItemList",
  "name": "キャリアタイプ別おすすめ転職・就活サービス一覧",
  "description": "AIキャリア診断で分かった仕事タイプに合った転職エージェント・就活サービス",
  "url": "https://career-dna.jp/jobs.html",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "【サービス名をjobs.htmlから転記】",
      "url": "https://career-dna.jp/jobs.html#【アンカーID】",
      "description": "【サービス説明をjobs.htmlから転記】"
    }
    // 実際の掲載サービス数だけ繰り返す
  ]
}
</script>
```

**期待成果物**
- jobs.html の `<head>` 内に ItemList JSON-LD 追加済み（掲載全サービスを正確に反映）
- `"numberOfItems"` フィールドに実際のサービス数を記載
- sitemap.xml の jobs.html lastmod を 2026-05-17 に更新
- コミット・プッシュ完了

**注意事項**
- jobs.html を必ず開いてサービス名・数を確認してから実装すること
- デザイン変更不要（head内のみの変更）

### タスク3：index.html に Organization JSON-LD 追加【新規】

**背景・理由**
index.html には WebSite JSON-LD（+ SearchAction）・WebApplication JSON-LD はすでに実装済み。
Organization スキーマ（Xアカウント sameAs・お問い合わせページ contactPoint）を追加することでGoogleのエンティティ認識・ナレッジパネル表示強化・AI検索エンジンへの認知向上が期待できる。

**実装内容**

index.html の `<head>` 内の既存 JSON-LD `<script>` タグの直後に以下を追加：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "@id": "https://career-dna.jp/#organization",
  "name": "キャリアDNA",
  "url": "https://career-dna.jp/",
  "logo": "https://career-dna.jp/images/ogp.png",
  "description": "AIが25問で強み・価値観・向いている仕事タイプを診断する無料AIキャリア診断サービス。転職・就活中の社会人・学生向け。",
  "inLanguage": "ja",
  "sameAs": [
    "https://twitter.com/career_dna_jp"
  ],
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "customer support",
    "url": "https://career-dna.jp/contact.html"
  }
}
</script>
```

**期待成果物**
- index.html に Organization JSON-LD 追加済み
- sitemap.xml の index.html lastmod を 2026-05-17 に更新
- コミット・プッシュ完了

**注意事項**
- 既存の WebSite・WebApplication・FAQPage・Speakable JSON-LDは削除・変更しないこと
- デザイン変更不要（head内のみの変更）
