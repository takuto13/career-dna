## 作業1：Search Console TXTレコード設定

### 実行してほしい日付
2026-05-07（今日）または2026-05-08（明日）の朝まで

### 作業内容
Googleサーチコンソールでcareer-dna.jpのドメイン所有権を確認するため、DNSにTXTレコードを追加する。

### 手順（詳細ステップ）

**ステップ1：Search ConsoleでTXTレコードを確認する**
1. [Google Search Console](https://search.google.com/search-console) にアクセス
2. 左上「プロパティを追加」をクリック
3. 「ドメイン」タブで `career-dna.jp` を入力し「続行」
4. 表示された `google-site-verification=XXXXXXXXXXXXXXXX` の形式のTXTレコードをコピーする（XXXXの部分は人によって異なる）

**ステップ2：ドメインプロバイダーのDNS設定でTXTレコードを追加する**
- career-dna.jp のドメインを管理しているサービス（お名前.com、ムームードメイン、Vercel Domainsなど）にログイン
- DNS設定画面を開く
- 「TXTレコード追加」から以下を設定：
  - **ホスト名（名前）：** `@`（ルートドメイン）
  - **タイプ：** `TXT`
  - **値（コンテンツ）：** Search Consoleで取得した `google-site-verification=XXXXXXXXX` の文字列
  - **TTL：** 3600（1時間）または自動
- 保存する

**ステップ3：Search ConsoleでDNS確認を実行する**
1. Search Consoleに戻り「確認」ボタンをクリック
2. DNS反映に時間がかかる場合（最大48時間）は翌日再実行
3. 「所有権を確認しました」と表示されればOK

**ステップ4：確認完了後にsitemap.xmlを送信する（同日または翌日）**
1. Search Console左メニューの「サイトマップ」をクリック
2. 「新しいサイトマップを追加」に `sitemap.xml` と入力して「送信」
3. 送信後「成功しました」と表示されればOK
4. サイトマップのURL：`https://career-dna.jp/sitemap.xml`（すでにpublicフォルダに存在）

---

### 理由
- Search ConsoleのTXTレコード設定はたくとにしか実行できない権限作業のため、CEOがたくとに依頼している
- インデックス登録完了が検索流入獲得の前提条件であり、これが完了しないとSEOの効果がゼロになる
- 1ヶ月後目標の「アフィリエイト初収益発生」のためにも、今週中にインデックス登録を完了させる必要がある
